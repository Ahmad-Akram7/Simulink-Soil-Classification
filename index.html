🟦 USCS Classification

ASTM D2487

The Unified Soil Classification System classifies fine-grained soils using plasticity characteristics derived from Atterberg limits. It uses the Casagrande plasticity chart to distinguish clays from silts.

Soils plotting above the A-line (PI > 0.73 × (LL − 20)) are classified as Clays (C), while those below are Silts (M). A Liquid Limit of 50 separates Low (L) from High (H) plasticity.

Decision Logic
IF % Passing #200 ≥ 50 → Fine-Grained Soil
  PI = LL − PL

  IF LL ≥ 50 → High Plasticity (H)
  ELSE       → Low Plasticity (L)

  IF plots ABOVE A-line → Clay (C)
  ELSE                  → Silt (M)
Output Classifications
Symbol Soil Type Description
CH High-Plasticity Clay Expansive, compressible — poor foundation material
CL Low-Plasticity Clay Moderate plasticity, common subgrade soil
MH High-Plasticity Silt Elastic, frost-susceptible
ML Low-Plasticity Silt Low plasticity, frost-susceptible

🟧 AASHTO Classification

AASHTO M 145

The AASHTO system assesses soil suitability for use as road subgrade material in pavement construction. It classifies soils on an A-scale from excellent (A-1) to very poor (A-7-6).

Groups A-1 through A-3 represent granular materials (excellent to good subgrade), while A-4 through A-7-6 are silt-clay materials with progressively poorer engineering properties.

The model uses percentage passing sieve #200, Liquid Limit, and Plasticity Index to evaluate boundary conditions for each AASHTO group sequentially.

Output Classifications
Group Description Quality
A-1 Stone fragments, gravel, sand ★★★★
A-2 Silty or clayey gravel/sand ★★★★
A-3 Fine sand ★★★★
A-4 Silty soil ★★★★
A-5 Silty soil (elastic) ★★★★
A-6 Clayey soil ★★★★
A-7-5 Clayey soil (elastic) ★★★★
A-7-6 Clayey soil (expansive) ★★★★

Key Concepts Glossary

Essential geotechnical terms used by both models.

🔬 Sieve #200
0.075 mm opening — separates fine-grained from coarse-grained soils.
💧 Liquid Limit (LL)
Water content at which soil transitions from plastic to liquid state.
🧱 Plastic Limit (PL)
Water content below which soil behaves as a semi-solid.
📊 Plasticity Index (PI)
PI = LL − PL; measures the range of plastic behavior.
📈 A-line
Boundary on the Casagrande plasticity chart separating clays (C) from silts (M).
🟦 USCS
Unified Soil Classification System — general geotechnical use worldwide.
🟧 AASHTO
American Association of State Highway and Transportation Officials standard.
🛣️ Subgrade
The native soil layer on which road pavement structure is constructed.
Quick Start

Get Started in Minutes

Clone, open, set inputs, and run.

1

Check Prerequisites

Make sure you have MATLAB R2023b+ with Simulink installed.

2

Clone the Repository

Download or clone the repo to your local machine.

3

Open a Model

Navigate to USCS_Model/ or AASHTO_Model/ and open the .slx file.

4

Set Inputs & Run

Double-click the Constant blocks to set your lab values, then hit ▶ Run.

# Clone the repository
git clone https://github.com/Ahmad-Akram7/Simulink-Soil-Classification.git
cd Simulink-Soil-Classification

% Open the USCS model
open('USCS_Model/USCS_Soil_Model.slx')

% Or the AASHTO model
open('AASHTO_Model/AASHTO_Soil_Model.slx')

% Set your inputs in the Constant blocks:
%   - PercentPassing200  (e.g., 60)
%   - LiquidLimit        (e.g., 55)
%   - PlasticLimit       (e.g., 28)

% Hit ▶ Run — classification appears in the Display block
